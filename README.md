# foreman-todo-api

A FastAPI-based TODO API built end-to-end by the [foreman](https://github.com/herbae-org/foreman) AI Factory as a demonstration.

Nothing in this repository is written by hand beyond this README, `CLAUDE.md`, and the pipeline scaffolding under `.github/`. Every endpoint, every test, every `pyproject.toml` line lands through the four-body-agent pipeline: Planner → Implementer → Fixer → Merger.

See `CLAUDE.md` for project scope and conventions. See [foreman](https://github.com/herbae-org/foreman) for the pipeline itself.
