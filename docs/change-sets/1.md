# Change-set for #1 — Scaffold the FastAPI project (health check + test harness)

## Scope

Bootstrap the minimal project skeleton: packaging metadata, the `src/todo_api/` package, a single `GET /health` endpoint, and a passing test. No TODO CRUD endpoints, no database, no persistence layer — those ship in later issues.

## Files to touch

- Create: `pyproject.toml` — PEP 621 metadata; `fastapi[standard]>=0.110` runtime dep; `dev` optional-dep group with `pytest` and `httpx`; `hatchling` build backend; pytest `testpaths = ["tests"]`; hatch wheel target pointing at `src/todo_api`.
- Create: `src/todo_api/__init__.py` — empty; marks package root.
- Create: `src/todo_api/app.py` — `FastAPI()` instance exported as `app`; single route `GET /health` returning `{"status": "ok"}` with HTTP 200.
- Create: `tests/__init__.py` — empty; lets pytest discover the package.
- Create: `tests/test_health.py` — one happy-path test using `fastapi.testclient.TestClient` asserting 200 and `{"status": "ok"}`; one failure-path test asserting a non-existent route returns 404.
- Create: `.gitignore` — Python standard ignores: `__pycache__/`, `*.pyc`, `.pytest_cache/`, `.venv/`, `dist/`, `build/`, `*.egg-info/`, `.coverage`.

## Public API / contract changes

Before: no HTTP surface exists.

After:
- `GET /health` → `200 OK`, body `{"status": "ok"}`

## Test plan

New tests (in `tests/test_health.py`):
1. `test_health_returns_ok` — `GET /health` → status 200, JSON body `{"status": "ok"}`.
2. `test_unknown_route_returns_404` — `GET /nonexistent` → status 404.

No existing tests to preserve (repo has none today).

Acceptance gate (run from repo root after `pip install -e .[dev]`):
- `python -c "from todo_api.app import app; print(type(app).__name__)"` prints `FastAPI`.
- `pytest -q` exits 0 with at least 1 passing test.

## Risk justification

Classified **high** because `pyproject.toml` introduces `fastapi[standard]` as a new runtime dependency, which is an explicit high-risk trigger in the rubric. The change also adds the first public HTTP endpoint (`GET /health`), which is a public API surface change.
